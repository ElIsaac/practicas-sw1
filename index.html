1. div

La etiqueta div se utiliza como contenedor para elementos HTML, que pueden ser estilizados con CSS o manipulados con JavaScript.

2. span

La etiqueta span se utiliza para agrupar elementos en línea en un documento, para que se les puedan aplicar estilos.

3. a

La etiqueta a define un hipervínculo, que se utiliza para enlazar de una página a otra.

4. img

La etiqueta se utiliza para incrustar una imagen en una página HTML.
